== Description ==

Converts Cyrillic characters in post slugs to Latin characters

= Standards =
1. GOST 16876-71
2. BSI
3. BGN
4. Yandex

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'AXP Cyrillic to Latin'
3. Activate BuddyPress from your Plugins page. (You will be greeted with a Welcome page.)

= From WordPress.org =

1. Download AXP Cyrillic to Latin.
2. Upload the 'AXP Cyrillic to Latin' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate BuddyPress from your Plugins page. (You will be greeted with a Welcome page.)
